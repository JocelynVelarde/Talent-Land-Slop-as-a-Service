Bienvenido a la documentación técnica y de proyecto de **Tecovolt**, un nodo inteligente de protección eléctrica construido sobre el Arduino Uno Q de Qualcomm para el track **Sustainable Power Cities** en Talent Land 2025.

Esta documentación cubre nuestra investigación, arquitectura técnica, modelos de Edge AI, hardware, análisis financiero y el journey completo del equipo durante el hackathon.
